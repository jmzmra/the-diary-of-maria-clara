# Noli Me Tangere: Interactive Diary of Maria Clara

This project is a **creative and educational digital reconstruction** inspired by José Rizal's classic novel *Noli Me Tangere*. It presents an interactive diary from the perspective of Maria Clara, capturing her inner thoughts, emotions, and reflections throughout key events in the story.

## Disclaimer

This project is for **educational purposes only**. It is intended to honor and acknowledge José Rizal’s work and provide an engaging way to explore the novel. 

- All content, design, and source code are **original creations** inspired by the novel.  
- This project is **not for commercial use** or redistribution.  
- Any resemblance to real persons, events, or institutions beyond the novel is purely coincidental.  
- All rights to the original novel remain with José Rizal and relevant copyright holders.

## Features

- Interactive digital book format with page-flipping animation.  
- Diary entries reflecting Maria Clara's perspective.  
- Sections for preface, table of contents, author’s notes, and contributions.  
- Styled with React and modern CSS for an immersive reading experience.

## Installation

1. Clone this repository:
```bash
git clone https://github.com/your-username/your-repo.git
cd your-repo
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the project locally:
```bash
npm start
# or
yarn start
```
